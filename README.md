# CabinPoker
Free Open source Poker Client for Cabinpoker.com


https://user-images.githubusercontent.com/122582335/212490714-c47c29c5-02f6-4a17-a36a-36c497ac3f3a.mp4


# Requirements
- NodeJS
- NPM

# Instructions
```git clone https://github.com/cabinpoker/client```

```cd client```

```npm install```

```npm start```
